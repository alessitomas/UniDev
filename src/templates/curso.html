Operadores

Os operadores são símbolos que realizam operações matemáticas. Os operadores aritméticos são os mais utilizados e são os seguintes:

  • + → Soma
  • - → Subtração
  • * → Multiplicação
  • / → Divisão
  • ** → Potenciação
  • // → Divisão inteira
  • % → Resto da divisão


Considere os exemplos a seguir:

print(2 + 3) print(7 - 3) print(4 * 3) print(8 / 2) print(3**3) print(5 // 4) print(7 % 5)
Saídas: 6 / 4 / 12 / 4 / 27 / 1 / 2




Importante lembrar!

    Em Python, existem dois tipos de números que são utilizados para realizar operações matemáticas:

  • int → números inteiros (sem nenhuma casa decimal)
  • float → números reais (pode possuir casas decimais)






Variáveis

São espaços na memória do computador que armazenam valores. Uma variável é como se fosse uma caixa com uma etiqueta com o seu nome. O conteúdo armazenado na caixa pode ser acessado a partir do nome em sua etiqueta.


Considere o exemplo a seguir:

a = 5 b = 10 c = a + b print(c)
Saída: 15

Estruturas de decisão

São estruturas que permitem que o programa tome decisões. As estruturas de decisão (if e else) são utilizadas para executar um bloco de código somente se uma condição for verdadeira.


Considere o exemplo a seguir:

a = 5 b = 10
if a > b: print("a é maior que b") else: print("b é maior que a")
Saída: "b é maior que a"

Estruturas de repetição

São estruturas que permitem que o programa repita um bloco de código várias vezes. As estruturas de repetição (while e for) são utilizadas para executar um bloco de código enquanto uma condição for verdadeira.



Usando a estrutura while, o bloco de código é executado enquanto a condição for verdadeira, o que acaba gerando a necessidade do que normalmente é chamado de contador, ou seja, um trecho de código que desempenhe a função de atualizar o valor que está sendo analisado quando o bloco for executado. Caso este não exista, o bloco de código será executado infinitamente, já que o valor não está sendo atualizado em nenhum momento.

Usando a estrutura for, a lógica é muito parecida com a do while. No entanto, neste caso, o contador é criado automaticamente pelo próprio Python. A função range é utilizada para criar uma sequência de números que será utilizada como contador, ou seja, o código será executado até quando o número de loops for igual ao número passado pelo próprio usuário nesta mesma função.



Considere os exemplos a seguir:

USANDO WHILE:
a = 0 while a < 5: print(a) a = a + 1

USANDO FOR:
for i in range(5): print(i)
Saída: 0 1 2 3 4




Dica!

    No lugar de "a = a + 1" no loop usando o método WHILE, é possível substituir por "a += 1", já que significam a mesma coisa e, portanto, só se diferenciam na maneira em que são escritos!

Funções

Funções são blocos de código que podem ser executados várias vezes. Elas são utilizadas para reutilizar código e organizar o código de maneira mais simples.

Para criar uma função, é necessário utilizar a palavra reservada def e, em seguida, o nome da função. Após isso, é necessário colocar os parâmetros que serão utilizados na função, caso haja. Por fim, é necessário colocar dois pontos (:) e, em seguida, o bloco de código que será executado quando a função for chamada.



Considere o exemplo a seguir:

def soma(a, b): soma = a + b return soma
print(soma(5, 10))
Saída: 15




Importante lembrar!

    Quando criamos uma função, é necessário que ao final do bloco de código seja utilizado a palavra reservada "return", que é utilizada para retornar o valor que foi calculado dentro da função. Caso não seja utilizado, o valor retornado será "None", que significa que não há valor algum sendo retornado pela função.

Listas

Listas são estruturas de dados que permitem armazenar vários valores em uma mesma variável. Elas são utilizadas para armazenar valores e organizar o código de maneira mais simples.

Para criar uma lista, é necessário utilizar colchetes ([ ]) e, em seguida, os valores que serão armazenados na lista separados por vírgulas (","). É possível armazenar qualquer tipo de dado dentro de uma lista, como números, strings, booleanos, etc.



Métodos importantes de lista em Python:

  • append(): adiciona um elemento no final da lista.
  • clear(): remove todos os elementos da lista.
  • copy(): copia a lista.
  • count(): conta quantas vezes um elemento aparece na lista.
  • extend(): adiciona os elementos de uma lista a outra.
  • index(): retorna o índice do primeiro elemento com o valor especificado.
  • insert(): adiciona um elemento em uma posição específica (índice).
  • pop(): remove o elemento de uma posição específica (índice).
  • remove(): remove o primeiro elemento com o valor especificado.
  • reverse(): inverte a ordem da lista.
  • sort(): ordena a lista.


Considere os exemplos a seguir:

lista = [1, 2, 3, 4, 5] lista.append(6)
print(lista)
Saída: [1, 2, 3, 4, 5, 6]




lista = [1, 2, 3, 4, 5] lista.pop(2)
print(lista)
Saída: [1, 2, 4, 5]




lista = [1, 2, 3, 4, 5] lista.reverse()
print(lista)
Saída: [5, 4, 3, 2, 1]




lista = [3, 2, 1, 5, 4] lista.sort()
print(lista)
Saída: [1, 2, 3, 4, 5]




Importante lembrar!

    Em uma lista, o primeiro item possui o índice 0. Logo, quando queremos nos referir ao terceiro elemento da lista, por exemplo, o índice de tal elemento seria o 2, já que o primeiro elemento possui o índice 0, o segundo possui o índice 1 e o terceiro possui o índice 2.

Dicionários

Os dicionários em Python possuem certa similaridade com a ideia das listas. A diferença é que, enquanto as listas são estruturas de dados que possuem índices numéricos, os dicionários possuem índices alfanuméricos, ou seja, no caso dos dicionários os itens dentro deste possuem um certo "rótulo" ( key ) que pode ser usado para que se retorne o resultado desejado ( value ).

Para criar um dicionário, é necessário utilizar chaves ({ }) e, em seguida, os valores que serão armazenados no dicionário separados por vírgulas (","). É possível armazenar qualquer tipo de dado dentro de um dicionário, como números, strings, booleanos, etc.



Métodos importantes de dicionário em Python:

  • clear(): remove todos os elementos do dicionário.
  • copy(): copia o dicionário.
  • fromkeys(): retorna um dicionário com as chaves e valores especificados.
  • get(): retorna o valor da chave especificada.
  • items(): retorna uma lista contendo uma tupla para cada par chave-valor.
  • keys(): retorna uma lista contendo as chaves do dicionário.
  • pop(): remove o elemento com a chave especificada.
  • popitem(): remove o último elemento inserido no dicionário.
  • setdefault(): retorna o valor da chave especificada. Se a chave não existir, insere a chave, com o valor especificado.
  • update(): atualiza o dicionário com os itens de um dicionário especificado.
  • values(): retorna uma lista de todos os valores do dicionário.


Considere os exemplos a seguir:

dicionario = {"nome": "João", "idade": 20, "cidade": "São Paulo"}
print(dicionario["nome"])
Saída: "João"




dicionario = {"nome": "João", "idade": 20, "cidade": "São Paulo"}
dicionario["nome"] = "Maria"
print(dicionario)
Saída: {'nome': 'Maria', 'idade': 20, 'cidade': 'São Paulo'}




dicionario = {"nome": "João", "idade": 20, "cidade": "São Paulo"}
dicionario["email"] = "joao@gmail.com"
print(dicionario)
Saída: {'nome': 'João', 'idade': 20, 'cidade': 'São Paulo', 'email': joao@gmail.com'}




dicionario = {"nome": "João", "idade": 20, "cidade": "São Paulo"}
dicionario.pop("nome")
print(dicionario)
Saída: {'idade': 20, 'cidade': 'São Paulo'}
